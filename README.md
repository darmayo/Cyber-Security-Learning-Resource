## 🎯 Purpose

Repo ini dibuat untuk mengumpulkan resource belajar cyber security secara rapi, supaya mudah dipakai untuk:

- Belajar dari basic sampai advanced
- Menyusun roadmap pribadi
- Latihan CTF dan vulnerable lab
- Belajar web pentest dan bug bounty
- Belajar Android/mobile security
- Belajar reverse engineering dan malware analysis
- Belajar SOC, SIEM, DFIR, blue team, dan threat intelligence
- Menyiapkan OSCP / CPTS / sertifikasi security lain
- Membangun portfolio cybersecurity

## 🧭 Suggested Learning Paths

### 1. Beginner Cyber Security

```text
Networking basic
→ Linux basic
→ Web basic
→ TryHackMe / Hack The Box Academy
→ OverTheWire
→ PicoCTF
→ PortSwigger Web Security Academy
```

### 2. Web Pentest / Bug Bounty

```text
OWASP Top 10
→ PortSwigger Web Security Academy
→ OWASP Juice Shop
→ Pentest Book
→ Bug bounty methodology
→ Hacker101 / YesWeHack DOJO
→ Practice reporting
```

### 3. SOC / Blue Team / DFIR

```text
Windows & Linux logging
→ SIEM basic
→ Splunk/Wazuh lab
→ Incident response notes
→ Threat intelligence
→ Malware triage basic
→ Detection engineering
```

### 4. Android / Mobile Security

```text
Android basic
→ OWASP MASVS / MASTG
→ APK static analysis
→ Android intent / storage / WebView issues
→ Dynamic analysis lab
→ Mobile API testing
```

### 5. Reverse Engineering / Malware Analysis

```text
Assembly basic
→ Static analysis
→ Dynamic analysis
→ Ghidra/radare2 basics
→ MalwareAnalysis101
→ DFIR workflow
→ Writeups and reporting
```
